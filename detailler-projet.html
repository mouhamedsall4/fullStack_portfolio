Détail du Projet

Site E-commerce

Description :

Un site de commerce électronique complet permettant aux utilisateurs de parcourir un catalogue de produits, d'ajouter des articles à leur panier et de procéder au paiement. Le site dispose d'un système d'authentification, d'une gestion des commandes et d'un tableau de bord administrateur.

Technologies utilisées :
  1. HTML5
  2. CSS3
  3. JavaScript
  4. React JS
  5. Express JS
  6. MongoDB
  7. Stripe API
Fonctionnalités :

• Authentification utilisateur
• Catalogue de produits
• Panier d'achat
• Système de paiement
• Gestion des commandes
• Dashboard administrateur
