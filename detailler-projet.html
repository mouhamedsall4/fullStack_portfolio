Detail du Projet

Site E-commerce

Description :

Un site de commerce electronique complet permettant aux utilisateurs de parcourir un catalogue de produits, d'ajouter des articles a leur panier et de proceder au paiement. Le site dispose d'un systeme d'authentification, d'une gestion des commandes et d'un tableau de bord administrateur.

Technologies utilisees :
  1. HTML5
  2. CSS3
  3. JavaScript
  4. React JS
  5. Express JS
  6. MongoDB
  7. Stripe API
Fonctionnalites :

Authentification utilisateur
Catalogue de produits
Panier d'achat
Systeme de paiement
Gestion des commandes
Dashboard administrateur
